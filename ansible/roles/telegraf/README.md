telegraf
==============

Ansible role to install and configure Telegraf.


## Example

```
- hosts: myhost

  vars:
    
  roles:
    - wunzeco.telegraf
```

> **Note:** 
>    example variables for configuring input and output plugins, see
>    `test/input-plugins-vars.yml` and `test/output-plugins-vars.yml` respectively

## Dependencies

none
